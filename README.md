MyVault - Voice Diary App. 

A professional voice diary application for Android built with Jetpack Compose and powered by Google's Gemini 2.0 AI.FeaturesCore Functionality

•Voice Recording: Record voice diary entries with visual feedback and duration tracking
•Audio Playback: Play back recordings with progress indicator
•Calendar View: Interactive calendar showing days with entries
•Timeline View: Chronological list of all diary entries with search•Entry Details: View individual entries with playback controls.

AI-Powered Features (Gemini 2.0)
•Transcription: Automatic voice-to-text transcription
•Summaries: AI-generated summaries of diary entries•Insights: Key themes and insights extraction•Sentiment Analysis: Emotional tone analysis
•Intelligent Search: Ask questions about your entries using natural language.
